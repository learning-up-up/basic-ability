# python(conda)配置

conda的清华镜像网站相对会更快
```
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/main
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/r
conda config --add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/msys2
conda config --set show_channel_urls yes
```

配置环境
```
conda create --name <name> python=<vision> -y
```


# pytorch

我们这里使用conda来进行安装，如果有NVIDIA的独立显卡，可以使用如下命令进行安装
```
conda remove pytorch torchvision pytorch-cuda cudatoolkit
```
对于wsl来说，则无需安装cudatoolkit，引文已经内置了驱动

这一切尽量在虚拟环境中进行，这样方便我们适配不同的版本，并且在更新时也不会使系统出问题。在删除时只需
```
conda remove -n myenv --all
```
